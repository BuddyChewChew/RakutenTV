# RakutenTV UK — IPTV Playlist & EPG

A combined M3U playlist and XMLTV EPG for **Rakuten TV UK**, updated automatically every 8 hours via GitHub Actions.

---

## How to use

Add the playlist URL directly to your IPTV client — the EPG is embedded in the header and will load automatically alongside the channels.

**Playlist (M3U):**
```
https://raw.githubusercontent.com/BuddyChewChew/RakutenTV/main/playlist.m3u
```

**EPG only (XMLTV) — if your client needs it separately:**
```
https://raw.githubusercontent.com/BuddyChewChew/RakutenTV/main/epg.xml
```

---

## How it works

The playlist and EPG are generated by combining data from two independent sources:

### 📺 Stream URLs
Stream URLs are sourced from the **[apsattv.com](https://www.apsattv.com)** Rakuten TV UK M3U playlist (`rakutentv-uk.m3u`), which provides working HLS stream links for each Rakuten UK channel.

### 📅 Programme Guide (EPG)
EPG data is fetched directly from the **Rakuten TV public API** (`gizmo.rakuten.tv/v3/live_channels`), providing up to 72 hours of programme information including titles, descriptions, and categories for each channel. This approach was originally developed by **[dp247](https://github.com/dp247/rakuten-uk-epg)** whose open-source project inspired and formed the basis of the EPG generation code used here.

### 🔗 Merging
The script matches channels from the M3U source to channels from the Rakuten API by name, then outputs a single playlist where each channel entry carries the correct `tvg-id` to link it to its EPG data. The EPG URL is embedded in the playlist header so compatible clients load both automatically.

---

## Update schedule

The playlist and EPG are refreshed automatically **every 8 hours** (00:00, 08:00, and 16:00 UTC). You can also trigger a manual update at any time from the [Actions tab](../../actions).
